# 航空驾驶舱模拟控制系统技术栈与实施方案

## 1. 项目概述

该项目是一个基于 C 语言和 SDL2 的桌面级飞行驾驶舱仿真系统，目标是模拟商用客机驾驶舱中的核心仪表与飞行管理逻辑，并通过网络协议接入 X-Plane 11 的实时飞行数据，实现高保真、可交互、可扩展的仿真展示。

## 3. 项目核心功能模块

### 3.1 基础框架

负责窗口初始化、资源加载、主循环、事件处理、配置文件读取与程序退出管理。

### 3.2 仪表盘渲染

实现 PFD、ND、EICAS 三大核心仪表盘的静态布局和动态刷新，包括姿态、速度、高度、航向、发动机参数和告警信息展示。

### 3.3 飞行数据接入

通过 UDP 从 X-Plane 11 接收实时飞行参数，并将数据解析后映射到各仪表盘显示层。

### 3.4 FMC 航路系统

实现飞行管理计算机的核心逻辑，包括航路编辑、航点管理、起降程序配置、航线保存与加载。

### 3.5 外部信息展示

通过 HTTP 接入高德地图 API，实现航班位置、航迹、天气等辅助信息展示。

### 3.6 多线程与性能优化

将渲染、网络通信和业务逻辑拆分到不同线程中，减少卡顿，提高刷新稳定性和响应速度。
